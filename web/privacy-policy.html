Privacy Policy – Acciacca Prof

Ultimo aggiornamento: 2026-02-19

Questa informativa descrive come l’app Acciacca Prof (“App”) gestisce i dati personali. L’App è progettata per funzionare in locale, senza creare account e senza inviare dati a server esterni.

1. Dati che l’App tratta

1.1 Foto selezionate dall’utente

  • Le foto selezionate vengono salvate localmente sul dispositivo.
  • Le foto non vengono caricate su server esterni.
  • Le foto possono essere esportate o importate manualmente dall’utente tramite file.

1.2 Preferenze e impostazioni

L’App salva in locale alcune preferenze (es. suoni, velocità, progressi di gioco) per garantire un’esperienza coerente.

1.3 Acquisti In‑App (IAP)

L’App può offrire un acquisto In‑App per sbloccare le personalizzazioni. La gestione dell’acquisto avviene tramite Apple App Store:

  • L’App non raccoglie né memorizza dati di pagamento.
  • Apple gestisce la transazione e i dati di pagamento.

2. Dati che NON vengono raccolti

  • Nessun account utente.
  • Nessuna profilazione.
  • Nessuna geolocalizzazione.
  • Nessun tracciamento pubblicitario.
  • Nessun invio di dati a server di terze parti.

3. Condivisione dati

L’App non condivide dati con terze parti. L’unica eccezione è l’eventuale condivisione volontaria di un file di esportazione delle personalizzazioni da parte dell’utente.

4. Esportazione/Importazione profili

Le personalizzazioni possono essere esportate in un file condivisibile. Questo file può contenere immagini; la condivisione è responsabilità dell’utente.

5. Sicurezza

I dati rimangono sul dispositivo. È responsabilità dell’utente proteggere il dispositivo stesso (es. PIN/Face ID).

6. Bambini

L’App non è progettata per raccogliere informazioni da minori. Non essendoci raccolta dati, non vengono trattate informazioni personali in modo intenzionale.

7. Modifiche alla policy

Questa policy può essere aggiornata. La data di aggiornamento è indicata in alto.
