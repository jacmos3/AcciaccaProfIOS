Acciacca Prof (iOS)

Avvio rapido in Xcode:
1) Crea un nuovo progetto iOS > App (SwiftUI) con bundle id `it.semproxlab.acciaccaprof` e iOS 15.
2) Sostituisci il contenuto della cartella del target con i file in `ios/AcciaccaProf/AcciaccaProf/`.
3) Aggiungi le risorse di `ios/AcciaccaProf/Resources/` in “Copy Bundle Resources”.
4) Assicurati che le immagini e i suoni siano nel bundle (Image1, Unit2_Image1, Unit4_Image1, profmorto.wav, ecc.).

Note:
- Il sottofondo MP3 non e' incluso; puoi aggiungerlo più avanti in `Resources/sounds/`.
- Le coordinate e dimensioni sono basate sul layout Delphi 618x543 e scalate automaticamente per iPhone/iPad.
